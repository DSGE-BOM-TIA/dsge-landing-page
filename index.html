Add-Value Instructions

Use this sequence to pull the system full circle from intake to monetization

Operational Playbook
1. Capture
Receive and identify material
Log source, weight, date, site, and contamination notes. Use the camera layer to classify the material before it is routed.
2. Verify
Confirm confidence and purity
If confidence is high, proceed automatically. If confidence is low, send the item to manual review so operators protect throughput and data quality.
3. Route
Select the highest-value path
Route to manufacturing when purity is high, resale when market value is favorable, and recycling when that is the safest and fastest value recovery path.
4. Monetize
Turn operations into measurable revenue
Show price per pound, processing margin, avoided disposal cost, compliance reporting value, and carbon or sustainability value where applicable.
If / then / else logic: If material confidence is 90% or higher and purity is 92% or higher, send to manufacturing. If purity is between 80% and 91%, evaluate resale versus recycling. Else hold for manual review or standard recycling. If carbon accounting applies, add ESG value to the route before final selection.
